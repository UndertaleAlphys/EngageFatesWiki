结合命运维基

Mod 安装教程

本文将介绍 结合命运 mod 的安装方法与常见问题说明。

一、运行环境要求

  • Ryujinx Emulator
    • 游戏：Fire Emblem Engage
    • 版本：v2.0.0
    • 请确保游戏本体可以在 Ryujinx 上正常启动并运行。

二、依赖组件

  • Cobalt
    • 推荐版本：v1.25.4 或更高
    • 本 Mod 基于 Cobalt 框架运行，低版本可能导致功能缺失或启动失败。

Cobalt 安装方式

  1. 下载 Cobalt 的 release 包。
  2. 解压压缩文件。
  3. 将 release 包中的所有内容直接拖入 SD 卡根目录。

无需进行额外配置。

三、文件结构与安装路径

本 Mod 由以下两个部分组成，二者缺一不可：

  1. IF mod (Cobalt)

    • Mod 本体

  2. IFmodSoundBank

    • 音乐与音效资源包

安装路径

请将以上两个文件夹完整放入以下路径： sd:/engage/mods/
├─ IF mod (Cobalt)/
└─ IFmodSoundBank/

四、常见错误与注意事项

1. 启动时报错

请依次检查：

  • Mod 是否安装在 正确的路径
  • 文件夹名称是否被修改
  • 是否遗漏 IFmodSoundBank

⚠️ 重要提示
请不要修改 IF mod (Cobalt) 文件夹内的任何内容。
插件在启动时会进行完整性检查，擅自修改可能导致 Mod 直接报错。

2. 与其他 Mod 的兼容性

  • 不建议与 修改 romfs 的 Mod 同时使用
  • 若需要使用其他插件：
    • 请新建 独立的 Mod 文件夹
    • 不要将文件混合放入 IF mod (Cobalt) 中

3. 关于实机（Nintendo Switch）运行

  • Switch 系统 不接受包含非英文字符的路径
  • 若在实机上使用：
    • 需要使用 特制的 Converter
    • 将路径转换为 仅包含英文与数字

否则系统将无法识别该 Mod。

如仍遇到问题，请再次确认安装路径与依赖版本是否正确。

五、关于压缩包格式的说明

  • 本 Mod 必须在解压后才能正常运行
  • 请不要直接将 .zip / .7z 等压缩包放入 mods 目录

尽管部分 Mod 在保持压缩包格式的情况下仍可能被加载，但 结合命运 不支持以压缩包形式运行。

请务必在安装前完整解压，并确认实际的文件夹结构如下： sd:/engage/mods/
├─ IF mod (Cobalt)/
└─ IFmodSoundBank/
