# 琳 ×7

## 属性加成

- **牵绊 1**：技+2  速度+2
- **牵绊 5**：技+4  速度+3
- **牵绊 10**：技+6  速度+4
- **牵绊 20**：技+8  速度+5

## 同步技能

- **牵绊 1**：强攻
  - 自身发动攻击时
若可追击且追击系数>=7
会抢先在敌人反击前施展追击
- **牵绊 5**：曲射
  - 可使用【曲射】指令
消耗4结合计量条或2结合回合
使弓或手里剑的射程+1
但命中减少20×(战斗距离-2)
- **牵绊 5**：先手必胜
  - 自身发动攻击时，
攻击力+4
- **牵绊 10**：强攻+
  - 自身发动攻击时
若可追击且追击系数>=6
会抢先在敌人反击前施展追击
- **牵绊 10**：速之吸收
  - 自身发动攻击时，每打倒1个敌人，速度+2
（至多叠加5层效果）
每回合开始时，若上回合没有参与战斗，
此效果减少1层
- **牵绊 20**：强攻++
  - 自身发动攻击时
若可追击且追击系数>=5
会抢先在敌人反击前施展追击
