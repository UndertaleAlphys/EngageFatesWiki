# 罗伊 ×5

## 属性加成

- **牵绊 1**：HP+3  力+2
- **牵绊 5**：HP+5  力+3
- **牵绊 10**：HP+7  力+4
- **牵绊 20**：HP+10  力+5

## 同步技能

- **牵绊 1**：坚持
  - 若战斗开始时HP为30%以上
受到超过HP的伤害时
防止额外的伤害
然后本回合坚持失效
- **牵绊 1**：地形耐性
  - 使地形伤害无效
不受地形回避和防御的负面影响
不受地形额外移动消耗的影响
- **牵绊 5**：四通八达
  - 可使用“四通八达”指令
朝合计4个方向的8个可选位置的敌人
前进至多2格并发动攻击
- **牵绊 5**：坚持+
  - 若战斗开始时HP为20%以上
受到超过HP的伤害时
防止额外的伤害
然后本回合坚持失效
- **牵绊 10**：乘胜追击
  - 对方处于不利地形时，
造成伤害+5。
- **牵绊 10**：坚持++
  - 若战斗开始时HP为10%以上
受到超过HP的伤害时
防止额外的伤害
然后本回合坚持失效
- **牵绊 20**：乘胜追击+
  - 对方处于不利地形时，
造成伤害+7。
- **牵绊 20**：坚持+++
  - 若战斗开始时HP为2以上
受到超过HP的伤害时
防止额外的伤害
然后本回合坚持失效

  - 单位等级+5

## 结合攻击

- **狮子红莲焰舞**
  - 使用炎之剑攻击前方
目标格与其后2格会陷入燃烧状态
仅限持剑且相邻时
- **狮子红莲焰舞（龙族）**
  - 使用炎之剑攻击前方
目标格与其后2格会陷入燃烧状态
仅限持剑且相邻时，【龙族】扩大燃烧范围
