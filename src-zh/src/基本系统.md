# 计算公式

- **命中**  
  `技 × 1.5 + 幸运 × 0.5 + 武器命中 + 支援命中`

- **回避**  
  `速度 × 1.5 + 幸运 × 0.5 + 武器回避 + 支援回避 + 地形回避`

- **必杀**  
  `(技 - 4) × 0.5 + 武器必杀 + 支援必杀`

- **必杀回避**  
  `幸运 × 0.5 + 武器必杀回避 + 支援必杀回避`

- **杖命中**  
  `魔力 + 技 + 武器命中`

- **杖回避**  
  `魔防 × 1.5 + 幸运 × 0.5 + 地形回避`

- **连携攻击伤害**  
  `敌方最大 HP 的 10%`

- **连携攻击命中率**  
  `100`

- **武器相克（Weapon Advantage）**  
  若单位在武器相克中占优势，则：
  - 命中 +20
  - 回避 +20
  - 造成伤害 +2
  - 所受伤害 -2

---

# 追击机制

- **追击系数**  
  默认为单位与敌方之间的**速度差**。  
  （部分武器可能会对该数值进行修正。）

- **追击条件**  
  当追击系数 **≥ 5** 时，单位可以进行追击。

---

# 结合气

- 若单位**主动发起战斗**或**击败敌人**：  
  → `结合气 +1`

- 若在同一次行动中**同时满足以上两项**：  
  → `结合气 +2`

---

# SP 系统

- 单位获得的 **SP = 获得的 EXP**  
- 是否装备纹章士戒指**不影响** SP 的获取
