# Lucina ×7

## Stat Bonuses

- **Lv 1**：Dex+2  Spd+1  Lck+2
- **Lv 5**：Dex+3  Spd+2  Lck+2
- **Lv 10**：Dex+4  Spd+3  Lck+4
- **Lv 20**：Dex+5  Spd+4  Lck+6

## Sync Skills

- **Lv 1**：Dual Strike
  - Unit participates in chain attacks as
if it were a backup unit.
- **Lv 5**：Dual Assist
  - When an ally attacks an enemy
within this unit's weapon range,
and if the unit is able to chain 
attack, join the chain attack.
- **Lv 10**：Dual Support
  - If unit is adjacent to allies, 
they take less Dmg. Value determined 
by the sum of support levels.
- **Lv 10**：Shuttle
  - Grants the unit "Shuttle" command,
use to swap places with an ally
within 5 tiles.
- **Lv 10**：Shuttle (Dragon)
  - Grants the unit "Shuttle" command,
use to swap places with an ally
within 5 tiles.
[Dragon] +2 range
- **Lv 20**：Dual Assist+
  - When an ally attacks an enemy
within this unit's weapon range,
and if the unit is able to chain 
attack, join the chain attack.
If this unit is at max HP,
deals 2x Dmg.

## Engage Skills

- **Lv 1**：Bonded
  - All allies within 2 radius gets the
skill "Dual Strike". For all units,
if he/she has skill "Dual Support",
the unit's also gets 1 support level
for each ally at distance 2.

## Engage Attack

- **All for One**
  - Use to make a sword attack against
a foe. All allies within 2
spaces chain attack.
